# Robot Quby [by Elia]

robot che crea tutto tranne olio e armi. arrivato dal suo pianeta con un atterraggio brusco quindi l'olio é andato disperso quindi chiede olio.
LOCATION: Foresta
VUOLE: TRE CARICHI DOLIO per aggiustare l'astronave.
